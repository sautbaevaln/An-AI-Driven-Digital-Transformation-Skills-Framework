# AI-Driven Digital Transformation Skills Framework — Machine Learning Analysis

This repository contains Python code for Section 4: Machine Learning Analysis of the AI-Driven Digital Transformation Skills Framework.

## Overview

This machine learning pipeline identifies digital transformation skills using Natural Language Processing and clustering techniques.

## Methods

The following methods were applied:

- Text preprocessing
- Skill extraction (TF-IDF and N-grams)
- Topic modeling (LDA)
- K-means clustering
- Hierarchical clustering
- PCA visualization

## Dataset

The dataset consists of Scopus-indexed articles including:

- Title
- Abstract
- Author Keywords
- Index Keywords

## Workflow

1. Data Preparation
2. Text Cleaning
3. Skill Extraction
4. Topic Modeling (LDA)
5. Skill Clustering (K-means & Hierarchical)
6. Visualization
